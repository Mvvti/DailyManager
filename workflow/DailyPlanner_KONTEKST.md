# Daily Planner — Kontekst projektu

## Opis
Boczny panel (sidebar widget) przyklejony do prawej krawędzi ekranu, zawsze widoczny, półprzezroczysty. Odpala się razem z Windowsem.

## Stack
- Electron + React + TypeScript + Tailwind
- Wzór: projekt WeatherApp (Electron + Vite + React + TS + Tailwind)

## MVP — 4 bloki
1. **Zegar + data** — wyświetlany na górze panelu
2. **Pogoda** — API pogodowe
3. **Lista zadań na dziś** — dodaj / odznacz / usuń, lokalny zapis
4. **Notatka dnia** — jedno pole tekstowe, auto-save

## Faza 2 (po MVP)
- Ciąganie notatek z LittleBrain
- Przypomnienia z powiadomieniami systemowymi
- Statystyki — ile zadań ukończono w tygodniu

## Sposób pracy
Budujemy etapami — jeden etap naraz, z gotowymi promptami dla Codexa.
